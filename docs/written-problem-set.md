# Written Derivation and Conceptual Question Bank

This question bank corresponds to the 20% "Written Derivation and Conceptual Questions" in the README. It supplements mathematics, complexity, experimental explanations, and engineering judgment that cannot be covered by automated programming tests. It is recommended to select 2-3 questions per chapter as formal assignments, with the remaining questions used for discussion sessions or final review; step-by-step recalculable small examples for each chapter can be found in the [Worked Example Pack](worked-example-pack.md).

## Grading Rubric

| Dimension | Score | Criteria |
|-----------|:-----:|----------|
| Mathematical Correctness | 35 | Correct derivation of symbols, conditional probabilities, matrix dimensions, gradients, or complexity |
| Shape and Boundaries | 20 | Clearly specify shapes for batch, sequence, head, vocab, mask, etc., and cover boundary conditions |
| Engineering Explanation | 20 | Ability to connect formulas with PyTorch implementation, memory, latency, and stability |
| Experimental Judgment | 15 | Ability to explain test outputs, failure cases, ablation, or metric limitations |
| Expression and Citation | 10 | Clear answer structure, cite papers/official documentation/model cards when necessary |

## Ch02 Embedding / Position Encoding / RoPE

## Ch03 Scaled Dot-Product Attention
