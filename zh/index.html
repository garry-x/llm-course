初学者怎么学

第一次学时，不要把所有高级内容都当成必修。先按主线跑通，再回头补高级结构和工程细节。

阶段章节目标建议
第一轮：主线必修Ch01-Ch06把文本变成 token，搭出能 forward 的 GPT编程练习必须写；DeepSeek 扩展先理解动机
第二轮：让模型跑起来Ch07-Ch08训练一个小模型，并让它生成文本重点看 loss、optimizer、sampling 的输入输出 shape
第三轮：高级内容选读Ch09-Ch10微调、对齐、RAG、推理优化和前沿架构先搭概念地图，再回头补公式和工程细节

最低前置要求：会写 Python 函数，理解 list/dict，知道基本矩阵乘法，能看懂 PyTorch tensor 的 shape。

课程知识系统

课程按“大模型如何被表示、计算、训练、生成、对齐和部署”来组织，而不是按工具名散排。

模块对应章节核心问题
表示层Ch01-Ch02文本怎么变成 token，token 怎么变成向量，模型怎么拿到位置信息
Transformer 核心Ch03-Ch06attention、mask、多头、GQA/MLA、Norm、FFN、MoE 和 GPT forward 怎么组合起来
训练循环Ch07loss、PPL、optimizer、checkpoint/resume 和分布式诊断怎么做
生成与推理Ch08-Ch10prefill/decode、采样、KV Cache、FlashAttention、RAG 和推理服务怎么取舍
微调与对齐Ch09SFT、LoRA、DPO、GRPO、RLVR/RFT 和 rollout 系统怎么改变模型行为
经典 NLP 与评估Ch11RNN/LSTM、依存句法、seq2seq、BERT 和传统指标怎么接到现代 LLM
