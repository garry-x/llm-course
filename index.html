课程主线

  • 表示与计算文本怎样变成 token，token 怎样进入 attention、block 和 GPT 前向传播。
  • 训练与对齐数据、loss、optimizer、checkpoint、SFT、偏好优化和 reward 假设如何共同决定行为。
  • 生成与服务采样、reasoning 预算、KV cache、RAG、tool use、压测和发布 gate 如何支撑线上系统。

系统化学习地图

基础知识不作为单独补课路径出现，而是在每章变成代码、公式、诊断表和工程判断。

01 文本接口

Ch01-Ch02 把 Python 数据结构、矩阵查表、点积和旋转位置编码放进 tokenizer、embedding、RoPE 与 token 成本。

02 Transformer 计算

Ch03-Ch06 把 softmax、mask、广播、归一化、残差路径、MoE 和 shape trace 组合成完整 GPT 前向传播。

03 训练闭环

Ch07 把 MLE、cross entropy、数据污染、optimizer state、checkpoint 和分布式成本变成训练 run 的最低证据。

04 生成与对齐

Ch08-Ch09 把 logits 分布、采样方差、KL/reference、SFT 数据协议、偏好数据、合成数据和 reward gate 放进 post-training。

05 推理系统

Ch10 把 KV cache、显存带宽、continuous batching、RAG/tool、长上下文、多模态和发布回滚统一到 SLO/成本账本。

06 评测与选型

Ch11 把 RNN、seq2seq、BERT、结构化预测、传统指标和 judge 可靠性用于任务建模、评测结论和 capstone 报告。
