Systematic Learning Map

Foundational knowledge does not appear as separate remedial paths but is embedded in each chapter as code, formulas, diagnostic tables, and engineering judgment.

01 Text Interface

Ch01-Ch02 embed Python data structures, matrix lookup tables, dot products, and rotary position encoding into tokenizer, embedding, RoPE, and token cost.

02 Transformer Computation

Ch03-Ch06 combine softmax, mask, broadcasting, normalization, residual paths, MoE, and shape traces into a complete GPT forward pass.

03 Training Loop

Ch07 turns MLE, cross entropy, data contamination, optimizer state, checkpoint, and distributed cost into the minimum evidence for a training run.

04 Generation & Alignment

Ch08-Ch09 embed logits distribution, sampling variance, KL/reference, SFT data protocol, preference data, synthetic data, and reward gates into post-training.

05 Inference System

Ch10 unifies KV cache, memory bandwidth, continuous batching, RAG/tool, long context, multimodality, and release rollback into SLO/cost ledger.

06 Evaluation & Selection

Ch11 uses RNN, seq2seq, BERT, structured prediction, traditional metrics, and judge reliability for task modeling and evaluation conclusions.
